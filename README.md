# YRcapture

윈도우용 캡처 도구입니다. 영역/창/전체 화면/웹페이지 캡처와 결과 편집(펜, 도형, 모자이크, 텍스트, 자르기)을 지원합니다. 캡처 이미지는 메모리에 보관되며, 저장 버튼을 눌러야 파일로 저장됩니다.

## 주요 기능

- 영역 지정(직접 지정) 캡처
- 창 캡처 / 전체 화면 캡처
- 웹페이지 전체 캡처(URL 입력)
- 결과 편집: 펜, 사각형, 화살표, 모자이크, 텍스트, 자르기
- 실행창(런처) + 결과창(미리보기) + 설정창
- 최근 캡처 목록, 개별 저장, 모두 저장, 모두 삭제
- 트레이 아이콘(좌클릭: 실행창 열기 / 우클릭: 메뉴)
- 단축키 설정(입력창 클릭 후 키 조합 입력), Print Screen 지원, 초기화

## 설치 및 실행

```bash
npm install
npm start
```

## 사용 방법

### 1) 실행창(런처)
- 직접 지정: 드래그로 영역 선택 캡처
- 창: 선택한 창 캡처
- 전체 화면: 현재 화면 전체 캡처
- 웹페이지: URL 입력 후 전체 페이지 캡처
- 결과창: 캡처 결과/편집 창 열기
- 설정: 단축키 등 옵션 변경

※ 실행창의 최소화 버튼은 작업표시줄로, 닫기 버튼은 트레이로 이동합니다.

### 2) 결과창(미리보기)
- 상단 도구로 편집(펜/사각형/화살표/모자이크/텍스트/자르기)
- 자르기: 파란 테두리/핸들을 드래그하면 영역이 조절되고, 마우스를 떼면 적용됩니다.
- 저장: 현재 이미지 저장
- 모두 저장: 폴더 선택 후 최근 캡처 전부 저장
- 모두 삭제: 메모리에 있는 최근 캡처 목록만 삭제

### 3) 트레이 메뉴
- 좌클릭: 실행창 열기
- 우클릭: 메뉴 표시(실행창/결과창/설정/백그라운드 종료)

## 단축키

### 기본 캡처 단축키 (설정에서 변경 가능)
- 전체 화면: Ctrl+Shift+1
- 창: Ctrl+Shift+2
- 영역: Ctrl+Shift+3
- 퀵 캡처: PrintScreen (퀵 모드는 설정에서 영역/창/전체 중 선택)

### 결과창 단축키
- Ctrl+Z: 되돌리기
- Ctrl+Y: 다시하기
- Ctrl+C / Ctrl+V: 복사 (클립보드)
- Enter: 텍스트 입력 확정
- Esc: 텍스트 입력 취소 / 자르기 드래그 취소

### 단축키 설정 방법
- 설정창에서 입력칸을 클릭한 뒤 원하는 키 조합을 누르면 자동으로 입력됩니다.

## 저장 정책

- 캡처 이미지는 기본적으로 메모리에만 저장됩니다.
- 저장 버튼을 눌러야 파일로 저장됩니다.
- 앱을 종료하면 저장하지 않은 캡처는 사라집니다.

## 폴더 구조 (요약)

```
src/
  main/        # Electron 메인 프로세스(윈도우/IPC/캡처)
  preload/     # Renderer - IPC 브리지
  renderer/    # UI(launcher, preview, settings, menu, webcapture)
  img/         # 앱 아이콘/트레이 아이콘
```

## 기술 메모

- Electron 기반 데스크톱 앱입니다.
- 웹페이지 캡처는 Chrome DevTools Protocol을 사용합니다.
- 결과 편집은 Canvas 기반으로 동작합니다.
